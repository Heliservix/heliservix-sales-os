# HeliserviX Sales OS

Proyecto comercial para prospección, CRM y automatización de correos para servicios de localización de cardúmenes de atún con helicóptero Robinson R44.

## Objetivo
Construir un sistema que:
- Identifique prospectos atuneros en Panamá, Ecuador y Pacífico Oriental.
- Registre empresas, contactos, oportunidades, contratos y seguimiento.
- Genere correos comerciales personalizados.
- Prepare automatización futura vía n8n + SMTP Hostinger Email.
- Ayude a cerrar contratos por faena o anuales para HP1804.

## Servicio principal
Localización aérea de cardúmenes de atún con helicóptero, incluyendo piloto profesional y mecánico.

## Lema
Eyes in the Sky. More Tuna in the Net.

Anticipo: USD 30,000

Modelo escalonado:
- 0–500 TN: USD 20/TN
- 501–700 TN: USD 40/TN
- 701–900 TN: USD 55/TN

Horas incluidas:

- Barcos mayores de 800 TN: 80 horas

Incluye:

- Mecánico

Excluye:
