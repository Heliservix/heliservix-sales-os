# API Application

The future API application will own:

- Tenant isolation.
- Authentication and authorization enforcement.
- Domain validation.
- CRM, intelligence, aircraft, contract, document, email, AI, dashboard, and report APIs.
- Background job orchestration.
- Integration adapters.
- Audit logging.

High-risk actions such as email sending, contract mutation, document export, and AI-proposed record changes must be explicit server-side commands.
