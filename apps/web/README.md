# HeliServiX Web

Version 0.1 of the HeliServiX Commercial Intelligence Platform web application.

This app is a Next.js, TypeScript, and TailwindCSS dashboard shell for the enterprise SaaS product. It establishes the UI architecture, navigation model, design language, and reusable components for future implementation work.

## Current Scope

- Professional responsive application layout.
- Sidebar and top navigation.
- Dashboard home.
- Module navigation for commercial intelligence, flight operations, documents, email, reports, and settings.
- Dark-mode-ready styling.
- Reusable dashboard and layout components.

Business logic, authentication, persistence, integrations, and API wiring are intentionally not implemented in Version 0.1.

## Local Development

```bash
npm install
npm run dev
```

The app runs on `http://localhost:3000` by default.
